# Darkside-Coffee

Full stack Coffee Shop application with:

- HTML, CSS, JavaScript frontend
- Node.js and Express backend
- MongoDB and Mongoose production database support
- JWT authentication
- Product catalog, cart, checkout, orders, contact, reviews, newsletter, and admin panel
- Render deployment configuration

Project code is inside `coffee-shop/`.

Render deployment guide:

```text
coffee-shop/RENDER_DEPLOYMENT.md
```
